# 💳 CreditWise - Loan Approval Prediction System

## 📌 Problem Statement
A mid-sized financial company, SecureTrust Bank, provides personal and home loans to customers across urban and rural regions. Currently, the bank relies on a **manual verification process** to evaluate loan applications based on income, employment details, credit history, and other factors.

This process is:
- Time-consuming  
- Inconsistent  
- Prone to human bias  

As a result:
1. Good customers sometimes get rejected → loss of business  
2. High-risk customers sometimes get approved → financial losses  

---

## 🎯 Objective
To build an **intelligent loan approval system** using Machine Learning that:
- Automates decision-making  
- Reduces human bias  
- Improves approval accuracy  
- Identifies high-risk applicants  

---

## 🧠 Machine Learning Approach
This is a **Supervised Learning (Classification)** problem.

Multiple classification algorithms were implemented and compared:

- Logistic Regression  
- Decision Tree  
- Random Forest  
- K-Nearest Neighbors (KNN)  
- Support Vector Machine (SVM)  

👉 The best-performing model was selected based on evaluation metrics.
